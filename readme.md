## Installation

Install webpack with yarn

```bash
yarn init
yarn add webpack webpack-cli webpack-dev-server -D
yarn add @babel/core @babel/preset-env babel-loader -D
yarn add css-loader style-loader -D
yarn add sass sass-loader -D
yarn add mini-css-extract-plugin -D
yarn add postcss postcss-loader postcss-preset-env -D

yarn add react react-dom
yarn add @babel/preset-react -D

yarn add cross-env -D

yarn add ts-loader -D
yarn add typescript -D

npx tsc --init
yarn add eslint -D
yarn add prettier -D


yarn add eslint-plugin-react@latest -D
yarn add @typescript-eslint/eslint-plugin -D
yarn add @typescript-eslint/parser -D
yarn add eslint-config-airbnb -D
yarn add eslint-plugin-import -D
yarn add eslint-plugin-jsx-a11y -D
yarn add eslint-plugin-react-hooks -D

yarn add eslint-config-prettier -D

yarn add husky -D//dont use

npx husky-init && yarn
yarn add lint-staged -D

yarn add stylelint stylelint-config-standard stylelint-config-prettier stylelint-order -D


//

```
